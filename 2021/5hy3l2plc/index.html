1. MATALB工具箱

SPM（Statistical Parametric Mapping）

  • 官网地址
  • 手册文档
  • 使用方法
    • 将下载文件解压并复制到MATLAB的toolbox文件夹下$\to$添加该文件夹及其子文件夹
    • MATLAB命令行界面输入：spm
    • 具体细节可以看安装文档

VBM Toolbox

VBM工具箱是SPM2，SPM5和SPM8的分割算法扩展的集合，以提供基于体素的形态测量（VBM，Voxel-Based Morphometry）。工具箱是根据SPM版本命名的，最新版本的更名为CAT12，它包含了对VBM8的实质性增强并且涵盖了多种形态学计量方法。

  • 下载地址
  • 手册文档
    • CAT12-Manual
    • VBM8-Manual
  • 使用方法
    • vbm系列工具基于MATLAB和SPM使用
    • 解压文件并复制到spm/toolbox路径下

xjview（可视化工具）

  • 下载地址
  • 手册文档（含视频）

REST

  • 下载地址
  • REST主要用于处理功能性磁共振数据，和VBM一样同样基于SPM和MATLAB，但是只需要复制REST文件夹到MATLAB的toolbox路径下

2. FSL

  • FSL（FMRIB Software Library）是用于FMRI，MRI和DTI脑成像数据的分析工具的综合库，大多数的工具都可以通过命令行或者图形用户界面操作。

  • 官网安装手册

    • 但是安装容易失败，这里提供一个Linux安装包，提取码：zimg
    • 官网的学习教程以及FAQs都很齐全，每个工具的介绍等也是如此，这里贴一个中文教程

  • 可处理数据类型广泛

    • 功能MRI、结构MRI以及DTI数据
    • 可统计分析结果：GLM/Stats

3. 查看图像及格式转换

MRIcro与MRIcron

  • MRIcron是MRIcro的进阶版本，它们都可以支持Win用户和Linux用户查看医学图像，它是一个独立程序。因此如果不是对MRIcro感情深或者内存不够，建议使用MRIcron，整如官网建议的：

    Note: MRIcro supports the Analyze file format. I strongly suggest using MRIcron. MRIcron supports the NIfTI format and has many new features. The only reason to use MRIcro instead of MRIcron is if you are familiar with this software or if you have a computer with limited memory (<256 Mb).

dcm2nii与dcm2niix

  • dcm2nii已过时。虽然我希望它总体上是健壮和有用的，但是积极的开发已经转移到dcm2niix。通常，dcm2niix通常更适合于现代DICOM图像。另一方面，dcm2nii可用于DICOM之前的传统专有格式的档案研究。dcm2nii（ DICOM to NIfTI conversion）是与MRIcron一起分发的独立程序，因此你可以在下载的MRIcron文件夹下找到对应的dcm2nii.exe（Win）或者dcm2nii（Unix）。
