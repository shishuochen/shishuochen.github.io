1. Termux的使用

Termux是什么

Termux 是一个Android 终端模拟器和 Linux 环境应用程序，无需 root 或设置即可直接运行，可以使用APT包管理器可以使用其他包（了解更多）。

安装及配置

国光大佬写的教程很是不错，推荐使用。教程包含了很多，按需所取，主要包括Hugo（如果你也用的这个），Git的安装。

2. 提交代码

很早之前就有：从 2021 年 8 月 13 日开始，在 GitHub.com 上对 Git 操作进行身份验证时，我们将不再接受帐户密码。（了解更多）。因此无论是从桌面端还是移动端都需要使用 Token 来验证身份，提交代码。

命令行实现

  1. 首先自然是创建库的Token

  2. 接下来就是正常的流程，只不过提交代码到GitHub前要获取验证（首次提交需要，之后就不用了）

    git init
git remote add origin https://<token>github.com/<github username>/<github repository name>.git
git add -A
git commit -m "Example"
git push -u origin master(如果不行就用强制，不过慎用 git push -f --set-upstream origin master)

放几个参考链接：链1、链2、链3。当然这些Google一下，StackOverflow上一大堆。
