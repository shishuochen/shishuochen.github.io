记录一下FSL（FMRIB Software Library）离线安装方法，也算是2023年的第一篇； 这篇文章主要参考的是CSDN上的一篇文章。

前言

关于FSL的介绍不再赘述了，能用到的朋友自然知晓其用处，下面主要说一下它的两种安装方法。一种就是官网手册提供的在线安装方法，这种方法的缺点就是慢、容易中途失败，这里我们介绍的一种以安装包的方式进行离线安装方法。

安装过程

下载安装包

文件地址： 这里（推荐使用IDM等工具下载）

解压文件到指定位置

一般来说，大家都把软件放在/usr/local或者/opt/路径下，以下是以前者为例展开介绍。

  1. 更改目标位置权限
    • 依次使用命令：cd /usr/local/ $\to$ sudo chmod -R 777 ./*
  2. 解压文件到指定位置
    • 首先cd到安装包所在目录
    • 输入命令: tar -zxvf fsl*.tar.gz -C /usr/local/

配置环境

  1. 打开/home/user/路径下，即~/.bashrc文件
    • 利用Vim打开： vim ~/.bashrc

  2. 在文件末尾添加以下内容

    export FSLDIR=/usr/local/fsl
export PATH=$PATH:$FSLDIR/bin
source $FSLDIR/etc/fslconf/fsl.sh

  3. 刷新配置文件

    • 终端输入：source ~/.bashrc

开始体验

至此，已经完成了安装配置等工作，接下里在终端输入fsl即可体验~
